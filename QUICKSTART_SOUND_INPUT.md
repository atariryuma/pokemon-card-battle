# クイックスタートガイド - サウンド＆インプット機能

## すぐに試す

1. **サーバーを起動**
   ```bash
   cd c:/Projects/pokemon-card-battle/pokemon-card-battle
   npm start
   # または
   node server.js
   ```

2. **ブラウザでアクセス**
   ```
   http://localhost:3000
   ```

3. **キーボードショートカットを試す**
   - `H`キーまたは`F1`キーを押してヘルプを表示
   - `M`キーでサウンドのON/OFF切り替え
   - `E`キーでターン終了（ゲーム中）
   - 矢印キーまたは`WASD`でナビゲーション
   - `Enter`または`Space`で確定

4. **ゲームパッドを試す**
   - ゲームパッド（Xboxコントローラー等）を接続
   - ゲームパッドのボタンを押すと自動認識されます
   - 右上に緑色のインジケーターが表示されれば接続成功

## 主な機能

### サウンド機能
- バトルBGMが自動再生（音源ファイルがある場合）
- カードアクションに応じた効果音
- `M`キーでミュート/アンミュート

### キーボード操作
- **基本**: `Enter`(確定)、`Esc`(キャンセル)、`E`(ターン終了)
- **移動**: 矢印キーまたは`WASD`
- **カード選択**: `1-7`で手札のカードを選択
- **UI**: `M`(サウンド)、`H`(ヘルプ)、`F`(フルスクリーン)

### ゲームパッド操作
- **Aボタン**: 確定
- **Bボタン**: キャンセル
- **Xボタン**: ターン終了
- **十字キー**: ナビゲーション

## オプション: 音源ファイルの追加

音源を追加したい場合は、以下の構造で配置してください：

```
public/
└── assets/
    ├── sounds/
    │   ├── card-draw.mp3
    │   ├── attack.mp3
    │   └── victory.mp3
    └── music/
        └── battle-theme.mp3
```

音源ファイルがなくても、ゲームは正常に動作します。

## トラブルシューティング

### サウンドが聞こえない
- ブラウザの音量を確認
- `M`キーでミュート解除
- コンソールでエラーを確認（F12でデベロッパーツールを開く）

### キーボードが反応しない
- ページがフォーカスされているか確認
- 入力フィールドにフォーカスしていないか確認
- ページをリロード（F5）

### ゲームパッドが認識されない
- ゲームパッドを接続後、ボタンを押す
- 対応ブラウザを使用（Chrome、Firefox推奨）
- コンソールで"Gamepad connected"メッセージを確認
